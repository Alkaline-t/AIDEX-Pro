AIDEX — Smart First Aid Box — Professional Package

Files:
- index.html      -> Main polished demo page
- styles.css      -> Professional CSS with animations
- app.js          -> JS interactions, accessibility, and simulated functions
- logo.svg        -> High-quality AIDEX logo (top-left)
- aidex-diagram.svg-> High-res placeholder diagram (replace with final artwork)

How to use:
1. Extract this folder.
2. Replace 'aidex-diagram.svg' with your final diagram image (same filename) if you have one, or keep placeholder.
3. Open index.html in a modern browser (Chrome/Edge/Firefox) to view the demo.
4. To deploy to GitHub Pages, push the folder to a repository and enable Pages from the main branch.
